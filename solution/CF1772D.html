题目传送门

思路

题目中提到，操作后的序列要满足 ai≤ai+1a_i \le a_{i+1}ai​≤ai+1​。所以，我们只需要比较相邻的两个数字即可。

当 ai=ai+1a_i = a_{i+1}ai​=ai+1​ 时，无论 xxx 取何值，都可以满足 ai≤ai+1a_i \le a_{i+1}ai​≤ai+1​。所以 xxx 的取值范围是 0≤x<+∞0 \le x < +\infty0≤x<+∞。

当 ai<ai+1a_i<a_{i+1}ai​<ai+1​ 时，要使 ∣ai−x∣≤∣ai+1−x∣|a_i - x| \le |a_{i+1}-x|∣ai​−x∣≤∣ai+1​−x∣，xxx 的取值范围为 0≤x≤⌊ai+ai+1⌋20 \le x \le \frac{\lfloor a_i+a_{i+1} \rfloor}{2}0≤x≤2⌊ai​+ai+1​⌋​。

当 ai>ai+1a_i>a_{i+1}ai​>ai+1​ 时，要使 ∣ai−x∣≤∣ai+1−x∣|a_i - x| \le |a_{i+1}-x|∣ai​−x∣≤∣ai+1​−x∣，xxx 的取值范围为 ⌈ai+ai+1⌉2≤x<+∞\frac{\lceil a_i+a_{i+1} \rceil}{2} \le x < +\infty2⌈ai​+ai+1​⌉​≤x<+∞。

遍历所有相邻的数字，计算 xxx 的每一个取值范围。如果有公共部分，则输出公共部分中任意一个数字即可。如果没有公共部分，则输出 −1-1−1。

时间复杂度 O(n)O(n)O(n)。
